## What this mod is

**Lanka Night Terrors** is a **single-player GTA V horror source mod** made for players who want the map to feel cursed after dark.

Instead of cheap nonstop jumpscares, this mod slowly breaks the player with:

- **night-only cursed activity**
- **sanity loss**
- **hallucination text warnings**
- **folklore ghost encounters**
- **weather escalation and dread-building ambience**
- **boss-like terror phases**

The mod is inspired by Sri Lankan folklore figures such as **Mohini**, **Maha Sona**, **Kalu Kumaraya**, **Riri Yaka**, **Bodilima**, and the **Devil Bird**.

---

## Features

### Core systems
- **Night Terror Engine** that becomes active after dark
- **Cursed Zones** placed in cemeteries, crossroads, church grounds, remote roads, and haunted-feeling locations
- **Sanity System** that drains when you remain in dangerous areas
- **Hallucination Messaging** that lies to the player and increases panic
- **Dynamic Weather Shifts** to thunderstorm conditions during high-stress events
- **Force Event Hotkey** for testing or cinematic recordings

### Folklore encounters
- **Mohini** — roadside lure encounter
- **Kalu Kumaraya** — shadow stalker pressure sequence
- **Bodilima** — sudden scream-and-attack event
- **Devil Bird** — omen-based atmosphere event
- **Riri Yaka** — blood-demon style assault phase
- **Maha Sona** — black dog intro that escalates into a final-form monster encounter

### Modder-friendly design
- **Plain source script format** using `.3.cs`
- **Editable INI config**
- **Easy to expand** with custom models, sounds, missions, or UI
- **Story Mode only** design philosophy

## Quick installation

### 1) Install required tools
You need:
- **Grand Theft Auto V (PC)**
- **Script Hook V**
- **Community Script Hook V .NET**

### 2) Add framework files to your main GTA V folder
Copy the ScriptHookV and ScriptHookVDotNet files into your **main GTA V directory**.

### 3) Create a `scripts` folder
If your game folder does not already contain a `scripts` folder, create one manually.

### 4) Drop in the mod files
Put these files inside the `scripts` folder:
- `LankaNightTerrors.3.cs`
- `LankaNightTerrors.ini`

### 5) Launch Story Mode
Start the game in **single-player Story Mode** and wait for the script to load.

---

## Detailed installation guide

### Step A — Install Script Hook V
1. Download **Script Hook V**.
2. Extract it.
3. Copy its required files into the **root GTA V folder**.
4. Do **not** use Script Hook V in GTA Online.

### Step B — Install Community Script Hook V .NET
1. Download **Community Script Hook V .NET**.
2. Copy these files into your GTA V root directory:
   - `ScriptHookVDotNet.asi`
   - `ScriptHookVDotNet2.dll`
   - `ScriptHookVDotNet3.dll`
3. Make sure your PC has the required **.NET Framework** and **Visual C++ Redistributable** installed.

### Step C — Install the mod script
1. Go to your GTA V folder.
2. Open or create the `scripts` folder.
3. Move the mod files into that folder.
4. Launch **Story Mode**.

### Step D — Confirm it loaded
When the game starts, you should see a notification that **Lanka Night Terrors** has loaded.

## Config file

You can edit `LankaNightTerrors.ini` to customize the experience.

### Default options
```ini
enabled=true
nightOnly=true
showZoneWarnings=true
sanityDrainPerTick=0.35
sanityRecoveryPerTick=0.22
eventMinSeconds=40
eventMaxSeconds=90
toggleKey=F10
forceEventKey=F11
```

### What these settings do
- `enabled` — loads the system in active mode
- `nightOnly` — only runs horror sequences during night hours
- `showZoneWarnings` — shows subtitle warnings when entering cursed areas
- `sanityDrainPerTick` — how fast sanity falls in danger zones
- `sanityRecoveryPerTick` — how fast sanity restores outside danger zones
- `eventMinSeconds` / `eventMaxSeconds` — random event timing range
- `toggleKey` — runtime mod toggle
- `forceEventKey` — testing shortcut

## Troubleshooting

### The mod does not load
- Confirm you are using **GTA V PC Story Mode**
- Confirm **Script Hook V** is installed correctly
- Confirm **ScriptHookVDotNet** files are in the game root folder
- Confirm the mod files are inside the `scripts` folder
- Confirm your script file name ends in **`.3.cs`**

### The game updated and scripts stopped working
This usually means Script Hook V or ScriptHookVDotNet needs a compatible update.

### No events are happening
- Wait until **night**
- Travel into the cursed zones
- Press **F11** to test event triggering
